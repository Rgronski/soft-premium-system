# Purpose

Project Brain jest centralnym silnikiem wiedzy (Knowledge Engine) platformy Soft Premium System.

Nie jest modułem UI.

Nie jest bazą dokumentów.

Nie jest generatorem plików.

Project Brain jest jedynym źródłem prawdy (Single Source of Truth) dla całego projektu.

Każdy moduł platformy odczytuje lub zapisuje dane wyłącznie poprzez Project Brain.

---

# Vision

Projekt powinien być zarządzany poprzez wiedzę, a nie poprzez pliki.

Markdown, PDF, README, ADR oraz inne dokumenty są jedynie reprezentacją wiedzy zapisanej w Project Brain.

---

# Responsibilities

Project Brain odpowiada za:

- przechowywanie wiedzy projektu
- relacje pomiędzy obiektami
- historię zmian
- kontekst AI
- decyzje architektoniczne
- roadmapę
- sprinty
- backlog
- historię wydań

Project Brain nie odpowiada za:

- wygląd aplikacji
- routing
- komponenty UI
- integracje zewnętrzne
- eksport dokumentów
